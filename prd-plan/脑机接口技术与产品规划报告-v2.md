## 一、行业概览与战略机遇

### 1.1 为什么是脑机接口

过去十年，人工智能在视觉、语言、决策等领域取得了突破性进展，但有一个方向始终被视为"终极交互"——**让机器直接读懂大脑**。脑机接口（Brain-Computer Interface, BCI）正是这一愿景的核心技术路径：通过对脑电信号的采集与解码，实现人脑与外部设备的直接交互，并可通过外部刺激形成闭环反馈调控机制，构成连接神经信号与智能系统的桥梁。

简单来说，BCI 做两件事：一是**读脑**——采集与解码特定大脑功能区产生的电信号，将其转化为控制外部设备的指令（脑→机）；二是**写脑**——将外部信息（如电刺激）反馈至大脑，实现对神经活动的调控（机→脑）。当这两条通路形成闭环时，就构成了极具价值的"感知-干预"一体化能力。

2026年的今天，这项技术正从实验室走向千家万户。**博瑞康的植入式 BCI 产品于今年3月获全球首发上市证，NeuraLink 持续推进人体试验，国内非侵入式消费产品已形成初步应用规模**——这不是一个"未来概念"，而是一个正在加速商业化的现实赛道。

脑机接口正在获得前所未有的政策支持力度。从国家战略定位、产业规划、医保准入到行业标准，一套完整的政策支撑体系已经形成：

- 科技部部长在"部长通道"明确表示，**"十五五"时期将加强脑机接口等领域的国家重大科技项目部署**

### 1.3 三条技术路线

BCI 技术按照信号采集方式的侵入程度，分为三个层级。每一层在信号精度与用户接受度之间做出不同的取舍：

| 类型 | 市场占比 | 采集方式 | 信号类型 | 典型应用场景 |
|------|---------|----------|----------|-------------|
| **非侵入式** | **75%** | 电极附着头皮 | 头皮 EEG | 睡眠管理、认知/情绪调节、外设控制 |
| **半侵入式** | — | 手术植入，未达皮层 | 皮层脑电 ECoG | 脑肿瘤术前功能定位、慢性疼痛管理 |
| **侵入式** | — | 电极植入大脑皮层 | 尖峰脉冲/局部场电位 | 重度瘫痪运动重建、帕金森深部脑刺激 |

非侵入式技术以其**零手术风险、用户门槛低、适配居家场景**的特点，占据了市场的绝对主体。这也正是我们选择的主赛道——不追求信号的极致精度，而是追求**规模化触达与持续性服务**的商业可行性。

> **我们的战略选择：** 聚焦非侵入式（EEG）赛道，面向睡眠管理、认知健康、情绪调节等居家场景，构建"硬件+AI+服务"的完整闭环。

### 1.4 市场规模与增长动能

从市场数据来看，BCI 产业已经越过"概念验证"阶段，进入了规模化增长的早期窗口：

- **全球市场：** 2025年达 **26.2亿美元**，预计2034年突破 **124亿美元**（CAGR ~19%）
- **国内市场：** 32亿元人民币，非侵入式产品占比高达 **75%~82%**

产业链层面呈现两个显著趋势：**上游核心部件国产化进程加速**（柔性电极、专用芯片等关键器件的自主可控能力持续提升）；**下游应用场景快速落地**，医疗康复与消费电子领域已形成可验证的商业闭环。

这意味着行业正处于从"技术驱动"向"市场驱动"转换的拐点。谁能率先在高频、刚需的居家场景中跑通产品闭环，谁就有机会占据这个百亿级市场的先发位置。

## 二、产业链全景与竞争格局

### 2.1 产业链结构

要理解我们在这个赛道中的战略卡位，首先需要看清整条产业链的结构。BCI 产业链可以清晰地分为上游核心部件、中游硬件设备与平台、下游应用场景三个层级。

**上游：核心部件**

上游是整个 BCI 系统的技术底座。两个核心部件决定了系统的信号质量与处理能力：

- **柔性电极** — 充当信号桥梁，既可采集电生理信号，也可实施精确的神经刺激。电极的材料科学与工艺水平直接决定信号的信噪比
- **主芯片** — BCI 系统的"前端大脑"，负责信号的实时采集、预处理与传输。低功耗、高采样率的专用芯片是实现便携化的关键

**中游：硬件设备与平台**

中游是产业链中竞争最为激烈、玩家最为多元的环节，涵盖 BCI 硬件终端、外部控制设备和数据采集平台三大品类：

| 品类 | 形态 | 代表企业 |
|------|------|----------|
| BCI 硬件设备 | 便携式头环 | 盈趣科技、迈雅（郑大）、强脑 BrainCo |
| | 额贴式设备 | 柔灵科技、云睿智能 |
| | 脑电帽 | 博瑞康、脑虎科技 |
| 外部控制设备 | 仿生手 | 强脑 BrainCo、傲意科技 |
| | 机器人 | NeuraLink、清华大学 |
| | VR/XR | Oculus |
| 采集平台 | 脑电采集系统 | 博瑞康、中电云脑、神念科技 |

### 2.2 下游应用：两大方向已形成初步规模

下游应用场景是 BCI 技术商业价值的最终承载。当前已形成医疗健康与商业消费两大方向：

**医疗健康——从临床向居家延伸：**

- **迈雅（郑大企业）** — 聚焦心理健康与儿童早期脑发展评估及训练，已建立"医院→居家"的服务模式，与协和等多家顶级医院开展临床合作
- **博瑞康** — 2026年3月其植入式 BCI 产品获全球首发上市证（NEO 系统），主要用于微创植入的脑电采集与治疗，标志着国产 BCI 医疗器械的里程碑突破
- **三博脑科** — 侵入式路线，聚焦人脑功能及疾病机制应用。与清华合作研发"北脑一号"，可在癫痫手术中实现高精度定位（准确率 >92%）

**商业消费——从极客向大众渗透：**

- **盈趣科技** — 测量大脑活动、追踪认知努力并优化睡眠，支持 EEG/PPG 多模态
- **华南脑控** — 多模态脑机接口，推出脑机 AI 鼠标、脑电轮椅控制系统等产品
- **傲意科技** — 元宇宙/VR 意念交互、肌电+脑电双模控制的游戏手柄

> **行业洞察：** 当前医疗级 BCI 正在快速获得临床验证和监管认可，而消费级 BCI 则在睡眠、专注力等高频场景中找到了产品-市场契合点。两个方向正在汇聚——**"医疗级精度 + 消费级体验"** 将成为下一阶段的核心竞争力标准。

## 三、产品规划

### 3.1 核心目标与产品哲学

我们的产品战略可以用一句话概括：**构建以非侵入式脑机接口（EEG）为核心，融合多模态生理数据（心电、体动等），为不同群体提供个性化、可干预的数字化健康解决方案。**

这里有三个关键词值得展开：

- **多模态融合** — 单一脑电信号存在天然的信噪比局限，通过融合心电、体动、呼吸等多维度生理数据，可以构建更完整的健康画像，显著提升识别准确率与干预效果
- **个性化** — 脑电信号的个体差异极大（这也是行业公认的技术难点），我们的技术路线特别强调"个体基线建立"和"小样本快速适配"，确保每个用户获得针对性的服务
- **可干预** — 不仅仅是"监测"和"报告"，而是形成"监测→识别→干预→反馈"的完整闭环。这是我们区别于市面上大多数脑电产品的核心差异化

### 3.2 切入策略与差异化定位

当前市场上的 BCI 消费产品普遍存在一个共同问题：**功能单一、场景孤立**——一款头环只做冥想，一款睡眠仪只做监测，一款训练器只做注意力。用户需要为不同需求购买不同设备，数据割裂、体验碎片化、健康管理缺乏连贯性。

我们的切入策略是在硬件、算法和产品/内容三个层面同时构建差异化优势，打造**"一机多能"的脑机健康平台**：

**一、硬件层：一机多能的多功能头环平台**

区别于市面上"一设备一场景"的单一产品形态，我们将脑电头环定位为**可扩展的健康入口平台**——同一款硬件设备，通过软件模式切换，即可适配睡眠管理、认知训练、情绪调节、注意力监测等多个场景：

| 差异化能力 | 说明 |
|----------|------|
| **硬件/软件模式切换** | 同一头环通过 App 切换"睡眠模式""认知训练模式""情绪管理模式"等，无需更换设备 |
| **多设备联动** | 头环作为核心节点，联动睡眠监护仪、安睡仪、睡眠灯、智能药盒等外设，形成场景化设备组合 |
| **开放硬件兼容** | 平台兼容第三方设备（手环、智能枕、戒指、毫米波雷达等），降低用户迁移成本 |

这种"一体机"的定位，使用户只需一次硬件投入，即可随产品迭代持续获得新的健康管理能力——类似智能手机对功能机的替代逻辑。

**二、算法层：个性化适配与跨场景迁移**

算法是我们最核心的技术壁垒。我们在两个方向上建立差异化：

- **个性化快速适配** — 行业痛点是脑电信号个体差异极大，通用模型效果差。我们基于跨被试迁移学习技术，实现新用户 **≤1分钟完成个性化校准**，解码精度提升 ≥10%。这意味着用户开箱即用，无需漫长的"训练期"
- **跨场景模型复用** — 基于数据增强和多任务学习，同一套底层模型可支撑睡眠分期、注意力检测、情绪识别等多个任务，不同场景之间的数据可以相互增强，形成**数据飞轮效应**

**三、产品/内容层：平台可配置 + 联合健康档案 + 个性化推荐**

产品层面的差异化集中在三个方面：

| 差异化能力 | 说明 | 竞品对比 |
|----------|------|----------|
| **平台可配置** | 运营侧可灵活配置健康方案（评估量表、训练内容、干预协议等），支持按人群、场景快速组合 | 竞品多为固化功能，无法灵活调整 |
| **联合健康档案** | 打通脑电、心电、睡眠、运动、用药等多维度数据，构建统一的个人健康档案，并支持家-医数据协同 | 竞品数据孤岛化，无法跨设备/跨场景关联 |
| **AI 个性化推荐** | 基于健康档案和持续监测数据，AI 智能体自动生成并动态调整干预方案（训练内容、助眠方案、用药提醒等） | 竞品多为静态方案，缺乏动态调整能力 |
| **内容生态** | 构建专业内容库（助眠音乐、冥想音频、认知训练课程、科普专栏等），支持持续扩展 | （待确认） |

> **总结：** 我们的差异化不是某个单点功能的领先，而是**"一机多能硬件 + 跨场景算法 + 平台化产品"三位一体的系统性优势**。竞品要复制这一模式，需要同时具备硬件集成、算法研发、平台运营和内容生态四种能力——这正是我们依托中国移动体系优势所构建的竞争壁垒。

### 3.3 目标用户分析

我们按照需求紧迫度、技术可行性和市场规模三个维度，将目标用户划分为四个群体，并确定了清晰的优先级排序：

| 优先级 | 目标人群 | 核心需求 | 上线计划 |
|--------|----------|----------|----------|
| **P0** | 睡眠障碍人群 | 睡眠质量评估、生理参数监测、助眠干预 | **6月** |
| **P1** | 中老年人（AD 风险群体） | 认知早筛、日常训练、情绪与睡眠监测 | **9月** |
| P2 | 儿童（3-12岁，发育期） | 语言训练、注意力与情绪训练、社交引导 | 12月（待定） |
| P2 | 焦虑/抑郁人群 | 情绪量化监测、压力评估、神经反馈训练 | 12月（待定） |

睡眠场景之所以排在首位，是因为它同时满足三个条件：**用户基数大**（中国成人失眠率超过38%）、**技术成熟度高**（睡眠分期是 EEG 研究最成熟的应用之一）、**闭环路径短**（从监测到干预的链路最清晰）。

### 3.4 核心设备矩阵

所有产品线共享一套核心设备矩阵，通过不同组合满足不同场景的需求：

- **脑电头环** — 核心入口设备。轻巧可穿戴，检测脑电、心率、体动等多模态信号，是四条产品线的通用基座
- **睡眠监护仪** — 非接触式放置于床头/床垫下，持续监测心率、呼吸、体动及在离床状态，适用于不愿佩戴头环入睡的用户
- **心电仪（手持单导联）/ 智能戒指** — 补充心脏健康数据，与脑电数据结合分析"心脑轴"状态，拓展监测维度

### 3.5 产品一：睡眠改善方案（P0，研发中，计划6月上线）

**目标人群与痛点：** 面向睡眠障碍患者，核心解决"入睡难、睡眠浅、无专业干预手段"的痛点。采用非侵入式技术，不依赖药物、无副作用，专为居家睡眠场景设计。

**硬件配置：** 脑电头环 + 睡眠监测仪 + 睡眠灯 + 安睡仪

**AI 应用功能全景：**

**一、睡眠评估体系**

| 功能 | 说明 | 状态 |
|------|------|------|
| 睡眠质量评估（PSQI） | 基于匹兹堡睡眠质量指数的标准化评估 | 已上线 |
| 失眠严重程度评估（ISI） | 量化失眠严重程度，支持干预效果追踪 | 规划中 |
| 心理健康评估 | 抑郁、焦虑等共病筛查 | 规划中 |

**二、睡眠数据监测**

我们构建了一套开放的多品类设备兼容体系——兼容智能手环/手表、智能枕、智能戒指、毫米波雷达等主流设备，同时提供**睡眠与慢病关联分析**能力，将睡眠数据与血氧、血压、血糖、心率、脑电等多维度数据进行交叉分析，发现隐性健康风险。当检测到心率异常或呼吸暂停等危险信号时，系统将即时推送预警通知。

**三、专业深度睡眠报告**

报告涵盖睡眠分期（深睡/浅睡/REM/清醒）、心率、呼吸、体动、鼾声、睡眠姿势及综合质量评分，通过可视化图表呈现，自动识别睡眠问题并给出针对性改善建议（如呼吸训练、作息调整等）。

更重要的是，我们引入了**"睡眠-日间表现关联分析"**机制：AI 健康管家在白天主动随访，通过对话式追问搜集用户日间行为数据，大模型进行根本原因分析与关联洞察，动态调整个性化睡眠改善方案，并联动设备实现效果改善。

## 四、核心技术能力

### 4.1 技术架构总览

我们构建了**五层技术架构**，自底向上逐层构建从数据到应用的完整能力栈。这一架构的设计逻辑是：底层解决"数据从哪里来"的问题，中间层解决"如何理解信号"的问题，顶层解决"如何服务用户"的问题。

```
┌─────────────────────────────────────────────────────────────────┐
│  应用层    睡眠管理 / 认知管理 / 行为干预 智能体                     │
├─────────────────────────────────────────────────────────────────┤
│  能力层    智能体引擎 + EEG 知识图谱 + 脑电大模型（待定）             │
├─────────────────────────────────────────────────────────────────┤
│  感知层    多通道时频融合 + 脑状态/意图识别                          │
├─────────────────────────────────────────────────────────────────┤
│  融合层    数据增强 + 跨模态迁移 + 模型泛化                          │
├─────────────────────────────────────────────────────────────────┤
│  数据层    Benchmark / DREAMT / UCI 基准数据集                      │
└─────────────────────────────────────────────────────────────────┘
```

### 4.2 数据层：高质量基准数据

数据是一切算法能力的起点。我们目前基于三套业界公认的高质量基准数据集开展模型研发：

| 数据集 | 来源 | 定位 | 规模 |
|--------|------|------|------|
| **Benchmark** | 清华大学（高上凯、高小榕、王毅军团队） | SSVEP 领域全球公认"金标准"，全球引用最高 | 64 通道，40 目标拼写器 |
| **DREAMT** | 2025年最新发布 | 最新睡眠分期基准 | PSG 多导图（EEG/EOG/EMG/ECG），100 位受试者整晚数据 |
| **UCI** | 公开基准 | 手势识别 sEMG 基准 | 36 人 × 7 种手势状态 |

### 4.3 融合层：解决脑电数据的"先天不足"

脑电数据面临三大先天挑战：**小样本**（标注成本极高）、**个体差异大**（同一任务不同人的脑电模式差异显著）、**跨设备/场景迁移困难**（实验室数据难以直接用于居家设备）。融合层正是为了系统性地解决这些问题而设计的。

| 技术方向 | 解决的问题 | 方法 | 状态 | 关键成果 |
|----------|----------|------|------|----------|
| **数据增强** | 标注数据不足 | EEG-GAN + 扩散模型 | **已完成** | 仅用 <5% 训练数据可恢复至 **88.07%** 精度；发表 ACM 会议论文 1 篇 |
| **跨被试适配** | 个体差异大 | 两阶段训练 + 领域自适应 | **研发中** | 基于清华 Benchmark 数据，个性化适配时间 **≤1min**，解码精度提升 **≥10%** |
| **多源信号融合** | 单模态信息不足 | 跨模态协同分析 | **研发中** | 已完成心电/音频/文本三模态融合；Q3 新增脑电/肌电，达到五类健康数据跨域协同 |
| **跨设备/跨场景** | 迁移困难 | 共享底层特征 + 自监督预训练 | 待定 | — |
| **模型泛化** | 通用模型构建 | 深度多任务学习 + 领域不变特征 | 待定 | — |

其中，**数据增强**方向的成果尤其值得关注：我们构建的基于噪声扩散的生成器模型，采用自注意力机制和条件嵌入的编码-解码架构，证明了在极端小样本条件下仍能保持高精度的可行性。这为后续居家场景中"冷启动"问题的解决提供了关键技术支撑。

### 4.4 感知层：让机器"听懂"大脑

感知层的核心任务是将原始脑电信号转化为有意义的状态判断。当前重点攻关两个方向：

**特征融合（研发中）：** 采用多通道 CNN、3D CNN、空间图神经网络（Spatial GCN）、动态图卷积网络等方法，实现多通道与时频域的联合特征提取，提升信号模式识别能力。

**已取得的关键指标：**

| 识别能力 | 精度 | 说明 |
|----------|------|------|
| 睡眠状态识别 | **93%** | 睡/醒二分类 |
| 睡眠分期识别 | **85.57%** | 深睡/浅睡/REM/清醒多分类 |
| 肌电运动意图识别 | **93%** | 基于在线学习，上肢四类运动模式 |

这些指标已达到支撑产品化应用的水平。特别是睡眠分期 85.57% 的准确率，已接近临床 PSG 人工判读的一致性水平（通常在 80-90% 之间），为我们的睡眠产品提供了可靠的技术基础。

### 4.5 能力层：从算法到智能体

能力层是连接底层算法与上层产品的关键中间件。我们在这一层构建三项核心能力：

**1. 智能体引擎（研发中）**

搭建即时反馈链路，基于实时数据监控和识别，实现个性化引导与调整。同时将脑机/神经干预技术 SKILL 化（如声光控制能力、冥想内容推荐能力等），构建周期任务唤醒架构，使智能体能够基于历史数据持续分析与自我优化干预方案，实现能力模块化与自主进化的一体化闭环。

**2. EEG 知识图谱（研发中）**

搭建包含模式、事件、脑区等信息的 EEG 知识图谱，在此基础上实现脑电数据和其他数据的联合分析。这一知识图谱将成为智能体进行推理和决策的知识底座。

**3. 脑电大模型（待定）**

传统脑电分析多为信号处理方法，缺乏跨场景泛化能力。2024年以 LaBram 为代表的 EEG 大模型、融合信号处理Encoder与注意力机制，为行业打开了新范式。我们计划基于感知层的自监督与对比学习技术，实现海量无标签脑电数据的通用特征表示，构建自有的脑电基础模型。

### 4.6 应用层：智能体驱动的健康服务

应用层聚焦焦虑性睡眠障碍、老龄化认知衰退、青少年心理健康等核心健康痛点，提供居家场景下的非侵入式干预与数字疗法：

| 智能体 | 核心技术支撑 | 计划上线 |
|--------|-------------|----------|
| **睡眠管理智能体** | 睡眠分期技术，包含睡眠质量评估、睡眠监测、改善方案制定等多个 SKILL 的闭环管理 | **6月** |
| **认知管理智能体** | 脑电特征 + 肌电意图识别，实现精准认知评估、注意力监测与互动式干预训练 | **9月** |
| **行为干预智能体** | 青少年综合发展评估、语言训练、社交引导、情绪与注意力训练等多 SKILL | 待定 |

## 六、移动战略定位与竞争优势

### 6.1 为什么中国移动要做脑机接口

这个问题的答案可以从两个维度来理解。

**从能力维度看：** 中国移动凭借强大的算网基础设施、亿级用户触点与泛在终端渠道，在脑机接口产业中天然具备成为**"全栈式解决方案集成商"** 的战略优势。运营商的网络覆盖、用户规模和渠道能力，恰好解决了 BCI 消费化过程中最大的瓶颈——**如何将前沿技术以低成本、大规模地触达终端用户**。

**从时机维度看：** 脑机接口技术正迎来向消费级渗透的拐点，特别是在睡眠管理闭环、慢病与康复监测、认知与注意力训练等居家场景中，展现出巨大的规模化推广潜力。这正是杭研及移动康养团队布局脑机赛道的核心阵地——可依托康养业务的长期积累，将前沿脑机交互技术转化为**有温度的居家健康服务**。

### 6.2 核心竞争力：四方协同飞轮

逐一拆解这四个环节：

**环节一：科研机构 — 底层技术创新**

当前顶尖高校及重点科研机构正集中攻坚 BCI 的底层机理与基础模型突破，包括研发通用脑电基础大模型，以及探索"脑电+肌电/红外"等多模态信号的高精度融合解码机制。它们构筑了行业的技术库，为产业化落地提供算法核心引擎与前沿理论支撑。

**环节二：杭研 — 工程转化与场景降维**

这是我们最核心的价值创造环节。将原本高度依赖专业人员、局限于医院或实验室的重度医疗检测（如多导睡眠图 PSG）及高算力科研模型，通过**算法轻量化、边缘部署**等工程手段，降维适配至居家消费级智能穿戴设备。从而真正实现在家庭环境中的普惠化、低门槛应用。

**环节三：杭研 — 数据优势与家医联通**

依托海量家庭用户与智慧家庭入口，我们具备持续获取**居家真实场景脑电数据**的独特能力——这一点是纯实验室研究机构和纯设备厂商都不具备的。更重要的是，通过聚合医院和家庭数据，实现数据深度增值与医养业务的无缝衔接，构建起数据层面的竞争壁垒。

**环节四：协同飞轮**

发挥运营商在真实场景数据、平台以及应用分发侧的体系化优势，承接并工程化科研院所的底层算法。同时，以海量真实的居家验证反馈数据反哺科研理论的演进。由此彻底打通**"真实场景数据获取 → 底层基础模型迭代 → 轻量化工程转化 → 居家康养应用验证"** 的闭环飞轮。

### 6.3 核心壁垒总结

| 壁垒 | 说明 | 竞争对手难以复制的原因 |
|------|------|----------------------|
| **数据飞轮** | 海量真实居家脑电数据持续反哺模型 | 实验室数据无法替代真实生活场景的多样性和规模 |
| **场景降维** | 医院级能力适配消费级设备 | 需要同时具备算法研发和工程化部署的复合能力 |
| **家医协同** | 聚合医院与家庭双端数据 | 运营商独有的用户触达能力和平台整合能力 |
| **渠道体系** | 亿级用户 + 泛在终端渠道 | 创业公司无法匹配的分发规模和用户信任基础 |
