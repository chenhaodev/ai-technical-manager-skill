# 第一章：AI医疗行业发展综述

> 本章为必读章节，涵盖行业全景、政策、市场规模等核心信息

## 1.1 产业全景图

### AI医疗应用领域
- **AI制药/AI医药研发**: 腾迈医药、康迈迪森、英矽智能、晶泰科技
- **AI医学影像**: 科亚医疗、数坤科技、汇医慧影、联影医疗、推想医疗、依图医疗、鹰瞳科技、医准智能、深睿医疗
- **AI医疗器械**: 微创机器人、精峰医疗、润达医疗、乐普医疗、连心医疗、柏视医疗
- **AI医疗助理**: 云知声、科大讯飞、灵医智惠、左手医生、腾讯健康
- **AI健康管理**: 医脉通、方舟健客、平安健康、壹健康
- **自动化药房**: 健麾信息、艾隆科技、蝶和科技

## 1.2 定义与分类

**AI医疗定义**: 以互联网为依托，通过基础设施搭建及数据收集，将人工智能技术及大数据服务应用于医疗行业，提升诊断效率及服务质量，解决医疗资源短缺、人口老龄化问题。

### AI+医疗健康生态架构
- **基础设施层**: 算力基础设施（芯片、服务器、云计算）、数据基础设施（数据采集、标注、医疗数据库）
- **模型层**: 算法研发（大模型、视觉大模型、多模态大模型）、模型管理维护
- **应用层**: 生命科学研究、药械研发、医学影像、医疗问答、辅助决策、医疗记录、个人健康管理

## 1.3 需求环境

### 医疗资源现状（2022）
| 指标 | 数据 |
|------|------|
| 全国医疗卫生机构总数 | 1,032,918家 |
| 医院总数 | 36,976家 |
| 三级医院 | 3,523家 (9.5%) |
| 三级医院诊疗人次 | 22.3亿 (58.4%) |
| 医师日均负担诊疗人次（医院） | 6.2人次 |

**核心问题**: 医疗资源分布严重不均，三甲医院集中于北上广和省会城市，承担全国58.4%的诊疗量。AI通过辅助导诊、问诊，有效提高诊疗效率，缓解医疗资源供需难题。

## 1.4 政策环境（2016-2023）

| 年份 | 政策 | 关键词 |
|------|------|--------|
| 2017 | 《新一代人工智能发展规划》 | 加快AI创新应用 |
| 2018 | 《关于促进"互联网+医疗健康"发展的意见》 | 互联网+医疗健康 |
| 2022 | 《"十四五"全民健康信息化规划》 | 促进医学AI应用试点 |
| 2023 | 《深化医药卫生体制改革2023年下半年重点工作任务》 | **加快推进**医学AI试点 |

**政策趋势**: 用词从"促进"升级为"加快推进"，鼓励AI医疗创新发展。

## 1.7 市场规模

### 中国AI医疗市场规模（2019-2028E）
| 年份 | 市场规模（亿元） | 占AI行业比重 |
|------|-----------------|--------------|
| 2023 | 107.0 | 8.6% |
| 2025E | 262.0 | 11.2% |
| 2028E | 976.0 | 15.4% |

**CAGR: 55.6%**

**增长驱动**:
1. 医疗资源分布不均，AI赋能医疗应运而生
2. 政策驱动医疗行业数智化发展
3. 大模型技术与生物制药、医学诊疗深度融合

## 1.8 市场前景

### 未来三年主要发力点
1. **AR与VR** - 65%计划部署
2. **视频分析** - 62%计划部署
3. **知识图谱** - 62%计划部署
4. **自然语言处理** - 60%计划部署

**趋势**: 从单点技术应用迈向多种AI能力融合，从事后分析迈向事前预判和主动执行。
